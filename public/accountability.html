📋 How This Tracker Works

Each pledge is sourced from documented BJP statements (manifesto, press conferences, rally speeches). Status changes only when official government orders, GO numbers, or independently verified action is reported by credible news sources. Campaign rhetoric is not treated as delivery. This tracker is updated as events occur — check back regularly.

Status Definitions

Watching — Promise made; no government action yet. Default state until an order is issued.

Announced — Government has formally announced the policy/scheme via official communication.

In Progress — Implementation has physically started (tenders issued, beneficiary lists activated, etc.).

Delivered — Independent verification confirms the promise has been substantially fulfilled.

Delayed — Past the stated or implied deadline without completion.

Broken — Government has explicitly reversed the policy or confirmed it will not be implemented.
